# BreathTimer
Приложение Андройд Cordova js<br>
Таймер дыхания, настроен на систему дыхания 1-2-1-2 <br>
Например Пауза (5 сек)- Вдох (10 сек) - Пауза (5 сек) - Выдох (10 сек)<br>
Компилированная версия в файле last.apk<br><br>
